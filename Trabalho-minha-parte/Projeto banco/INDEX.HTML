Quais documentos são necessários para solicitar o empréstimo?

Se você optar pelo empréstimo pessoal deve preencher o formulario com os dados solicitados, e anexar os documentos: IDENTIDADE,CPF,COMPROVANTE DE ENDEREÇO E COMRPOVANTE DE RENDA
Se você optar pelo emprestimo juridico deve preencher o formulario com os dados solicitados, e annexar os documentos: CARTAO CNPJ, DECLARAÇÃO ANUAL DO ANO ANTERIOR,CND FEDERAL

Qual o prazo para receber meu dinheiro?

A partir do momento que você é aprovado, o prazo médio para o dinheiro ser depositado na conta cadastrada é de no máximo, 3 dias úteis.
